I.
1.

FACTS

On December 10, 2021, Tree Farm filed its Request for Agency Action in this

matter (the “Request”). The matter was set to be heard by the Board at its January hearing.
2.

On January 10, 2021, the Division filed its response to the Request (the

“Response”).
3.

Also on January 10, 2021, Millcreek filed a Motion to Intervene in the proceeding

(the “Motion to Intervene”).
4.

Tree Farm is permitted to file a reply to the Response, Utah Admin. Code R641-

105-200, and is permitted to file a motion to amend the Request “at or before the hearing.” Utah
Admin. Code R641-104-240.
5.

Tree Farm is permitted to file a response to the Motion to Intervene “at or before

the hearing.” Utah Admin. Code R641-103-120.
6.

Rather than having Tree Farm file papers “at or before the hearing,” and in the

interest of giving all participants, including the Board, adequate time to fully consider the filings
that will be made in this matter, Movants hereby file this Stipulated Motion to Continue Hearing
and Establish Briefing Schedule, according to the following schedule:
a. January 26, 2022: Deadline for Tree Farm to file its reply to the Response or
a motion to amend the Request together with an amended request for agency
action, and its response to the Motion to Intervene.
b. February 10, 2022: Deadline for the Division to file its response to any
amended request for agency action (if any), and for Millcreek to file any reply
to Tree Farm’s response to the Motion to Intervene.
