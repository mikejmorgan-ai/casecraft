## CLAIM 1: DECLARATORY RELIEF — ORDINANCE INVALID (17-41-402)

**Legal theory:** The ordinance prohibits extraction of "sand, gravel and/or rock aggregate" — the statutory definition of Critical Infrastructure Materials (CIM). Utah Code 17-41-402(6) expressly prohibits counties from adopting ordinances that "prohibit" CIM operations. The ordinance is therefore preempted and void ab initio.

## CLAIM 3: DECLARATORY RELIEF — VESTED MINING RIGHTS (17-41-501 to -503)

**Legal theory:** Tree Farm has a vested mining use under Utah Code 17-41-501 (now 17-81-401), which provides that mining uses are "CONCLUSIVELY PRESUMED" vested if the use predates county zoning. Mining on this property dates to the 1890s — before Utah statehood (1896) — and decades before Salt Lake County adopted zoning. The county cannot extinguish vested rights by ordinance.
