# Claim 1: Ordinance Invalid -- State Preemption Under Utah Code 17-41-402(6)

> **Legal Theory:** The Salt Lake County ordinance eliminating mineral extraction as a
> conditional use and explicitly prohibiting sand, gravel, and rock aggregate extraction
> is facially invalid because it directly conflicts with Utah Code 17-41-402(6), which
> prohibits counties from adopting ordinances that restrict critical infrastructure
> materials operations.
